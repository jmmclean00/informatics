Mini Project Three

This application uses either a Yolo or Vision Transformer Model to predict the objects within an image.


Description of Models:

Yolo - detects and localizes objects within images by dividing the image into grid cells and predicting bounding boxes and class probabilities for each cell.

Vision Transformer - divides pictures into smaller parts and determines how these parts relate to each other in order to analyze the content within the images.
